# Baby Cry Interpreter AI 👶🍼

AI-powered mobile application to detect and interpret baby cries (Hungry, Sleepy, Uncomfortable, etc.) using Gemini 3 Flash.

## Tech Stack
- **Mobile:** Flutter
- **AI Model:** Gemini 3 Flash (OpenAI Compatible API)
- **Backend/Proxy:** Node.js (Optional Bridge)

## Features
- Real-time audio recording
- Multimodal AI analysis (Native Audio Understanding)
- Actionable insights for parents

## Setup
1. Clone the repository
2. Copy `.env.example` to `.env` and fill in your credentials
3. Run `flutter pub get`
4. Run `flutter run`
